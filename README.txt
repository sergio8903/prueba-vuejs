Pasos

1. Alojar carpeta bd en un servidor local Php.
2. Importar la base de datos vue.sql.  
   La Bd se nombra pruebavue.
   tabla vue.
3. Alojar el proyecto vue en el localhost.
4. Abrir un cmd y colocar la ruta donde esta el proyecto de vue.
5. Ingresar el comando npm install para instalar las dependencias.
6. Ingresar el comando npm run serve para iniciar el servicio.
